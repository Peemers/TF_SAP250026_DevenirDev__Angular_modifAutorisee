Démonstration 06 - Les pipes (intégrés)

Syntaxe:

{{ expression | nomPipe }}

Exemples

Chaînes de caractères:

  • lowerCasePipe: {{ "couCou Les petitS LOUpS" | lowercase }}
  • upperCasePipe: {{ "couCou Les petitS LOUpS" | uppercase }}
  • titleCasePipe: {{ "couCou Les petitS LOUpS" | titlecase }}

Nombres:

  • currencyPipe: {{ 1234.56 }}
  • currencyPipe: {{ 1234.56 | currency }}
  • currencyPipe: {{ 1234.56 | currency : 'EUR' }}
  • currencyPipe: {{ 1234.56 | currency : 'JPY' }}
  • currencyPipe: {{ 1234.56 | currency : 'EUR' : 'symbol' }}
  • currencyPipe: {{ 1234.56 | currency : 'EUR' : 'symbol-narrow' }}
  • currencyPipe: {{ 1234.56 | currency : 'EUR' : 'code' }}
  • currencyPipe: {{ 1234.56 | currency : 'EUR' : 'symbol' : '6.1-1' }}
  • currencyPipe: {{ 1234.56 | currency : 'EUR' : 'code' : '1.2-2' : 'fr-BE' }}
  • currencyPipe: {{ 1234.56 | currency : 'JPY' : 'symbol' : '' : 'ja-JP' }}

Pareil pour DecimalPipe et PercentPipe. (plus simple: digitsInfo + locale)

Date:

  • datePipe: {{ now }}
  • datePipe: {{ now | date }}
  • datePipe: {{ now | date : 'dd/MM/yyyy HH:mm:ss' }}
  • datePipe: {{ now | date : 'short' }}
  • datePipe: {{ now | date : 'dd/MM/yyyy HH:mm:ss' : '+0230' }}
  • datePipe: {{ now | date : 'EEEE dd MMMM yyyy HH:mm:ss' : '' : 'ja' }}

JSon:

  • jsonPipe: {{ personne | json }}
